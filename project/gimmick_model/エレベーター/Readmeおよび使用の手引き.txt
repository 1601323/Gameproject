エレベーターモデル
制作者：必殺仕事P（twitter：HissatuShigotoP）

このモデルはＰＭＸとなっておりますので、ＰＭＸモデルが使用できるバージョンのＭＭＤにてお使いください。
そぼろさんのAutoLuminous2に対応しています。（光らせるときはセルフシャドウオフを忘れずに！）

ややモーフが複雑なので簡単に解説。

「目」に登録されている「〜階」のモーフは対応する各階のボタンが光ります。
「リップ」に登録されている数字はエレベータの階数表示が対応する数値になります。
「まゆ」に登録されているものは以下の通り。
　　緊急停止：緊急停止表示が点灯します。
　　上昇・下降：エレベータ内に上昇・下降を表示が点灯します。
　　開・閉；エレベータ内の開閉ボタンが点灯します。
　　スイッチ上・下：エレベータ外の上昇・下降ボタンが点灯します。
　　到着上・下：エレベータ外の到着表示が点灯します。
　　天井照明：天井の照明が点灯します。
「その他」に登録されているものは、エレベータ内の7セグ表示それぞれに対応しています。
　階数は30階まで登録されていますが、99階まで表示が可能です。
※いずれもAutoLuminous2を適用しないと点灯しません。

このモデルの改造は可、再配布は不可とします。
また、公序良俗に反する動画への使用は禁止します。
